# Code of Conduct

## Our Pledge
We promise to be friendly and respectful to everyone.

## Our Standards
**Good behavior:**
- Be kind to others
- Help each other learn
- Give helpful feedback

**Bad behavior (not allowed):**
- Mean or rude comments
- Sharing private information
- Any kind of harassment

## Enforcement Responsibilities
Project leaders will make sure everyone follows these rules.

## Scope
These rules apply everywhere in this project.

## Enforcement
If someone breaks the rules:
1. First: We talk to them privately
2. Second: Warning
3. Third: Temporary ban
4. Finally: Permanent ban

## Enforcement Guidelines
We will be fair when enforcing rules.
